lsfile - A light macOS command line folder look up/management tool

This project is for make look and change files in folders easier.
Features :
show what's in the folder(Of course)
show things in Finder
copy a path
create a symbol link to the path copied
run shell without exit (beta)

Please help me add these features if you can:
make it show things like "less" (full screen)

depends:
macOS, "open" command
/bin/zsh
ls
dirname
basename
realpath

lsfile - 一个轻量 macOS 命令行目录查看/管理工具
本项目为了提供更简便的命令目录件操作体验而生
功能：
显示文件夹内容(还能没有吗）
在 访达 中显示
拷贝一个地址
创建一个软连接到拷贝的地址
在软件内运行shell (shell)


期望的功能：
让其向“less”一样显示输出 (全屏模式)

依赖：
macOS, "open" command
/bin/zsh
realpath
ls
dirname
basename
realpath
